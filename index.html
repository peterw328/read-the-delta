Read the Delta

Tracking how key U.S. economic data changes over time, release by release.

Coverage

  • Jobs Report Monthly employment, unemployment, and wage data.
  • Inflation Consumer price changes.

Additional indicators will be added as they are standardized.

How to Use This Site

  • This is a reference tool. It describes what changed, not why.
  • No forecasts, predictions, or investment advice.
  • Data comes directly from official government sources.
